##  Uyuni API Migration Tracker

A lightweight dashboard to track the migration of Uyuni XML-RPC APIs to OpenAPI/Swagger.  
It provides a structured view of API methods across namespaces, including status, HTTP mapping, and complexity.

##  Features
1. Search, filter, and sort API methods  
2. Real-time migration progress visualization  
3. Namespace-wise tracking (65+ namespaces)  
4. Highlights complexity to prioritize migration
